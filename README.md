# SlideOutMenu Repo

This is the source for the tutorial on how to [Make a SlideOut Menu](https://www.iamcarlos.com/slideout-menu-part-1).

## Getting started

1) Clone this repo
```
> git clone git@github.com:carlosmontes/SlideOutMenu.git
```

2) Install Pods
```
> cd SlideOutMenu
> pod install
```

3) Open workspace
```
> open SlideOutMenu.xcworkspace
```

4) Run the project
